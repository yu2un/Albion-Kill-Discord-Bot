# Albion-Kill-Discord-Bot
Albion Online Kill Discord Bot


### Installation

Albion Kill Bot requires [Node.js](https://nodejs.org/) v4+ to run.

```sh
discord.js
request
merge-images
canvas
fs
fs-extra
image-downloader
```
